## How to use `amalitech-icon` Components

<div style={{
    marginTop: '30px',
}}></div>

### `linear`

```html
<amalitech-icon-alarm></amalitech-icon-alarm>
```

### `bold`

```html
<amalitech-icon-alarm-bold></amalitech-icon-alarm-bold>
```

### `two-tone`

```html
<amalitech-icon-alarm-two-tone></amalitech-icon-alarm-two-tone>
```

### `bulk-two-tone`
    
```html
<amalitech-icon-alarm-bulk-two-tone></amalitech-icon-alarm-bulk-two-tone>
```

## Exposed CSS Variables

```css
amlitech-icon-alarm {
  --alarm-icon-color: #000;
  --alarm-icon-width: 24px;
  --alarm-icon-height: 24px;
}
```
