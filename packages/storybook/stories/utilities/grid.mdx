# Grid

You have up to 12 columns in your grid.

Each column has a 1/12 width by default. You can change the width of a column by adding a class `col-` followed by the number of columns you want to use.
all the column classes are appended with media queries for the coresponing screen size.

## Example
For a 3 column layout on large screens, 4 column layout on medium screens and 6 column layout on small screens, you can use the following code:
```html
<div class="row grid-gap-2">
    <div class="col-lg-3 col-md-4 col-sm-6">Column 1</div>
    <div class="col-lg-3 col-md-4 col-sm-6">Column 2</div>
    <div class="col-lg-3 col-md-4 col-sm-6">Column 3</div>
</div>
```
