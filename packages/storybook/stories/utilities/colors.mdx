## Using the Colors

<br />

#### Using colors with text 
```html
<div>
  <p class="text-red-100">Some text here</p>
</div>
```

#### Applying colors on text hover
```html
<div class="text-hover-green-200">
  <p>Some text here</p>
</div>
```

#### Using colors with backgrounds 
```html
<div class="bg-green-600">
  <p>Some text here</p>
</div>
```

#### Applying colors on background hover
```html
<div class="bg-hover-green-400">
  <p>Some text here</p>
</div>
```

#### Using colors with border
```html
<div class="border-green-400">
  <p>Some text here</p>
</div>
```

#### Applying colors on border hover
```html
<div class="border-hover-green-400">
  <p>Some text here</p>
</div>
```

#### Using colors with outline
```html
<div class="outline-green-400">
  <p>Some text here</p>
</div>
```

#### Applying colors on outline hover
```html
<div class="outline-hover-green-400">
  <p>Some text here</p>
</div>
```
