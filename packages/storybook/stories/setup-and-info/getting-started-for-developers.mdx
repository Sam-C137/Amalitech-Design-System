# Setup for Developers

Amalitech Design System provides frontend developers and engineers with a collection of reusable web components to build websites and user interfaces in major frontend frameworks like Angular, and HTML.

Adopting the design system enables developers to use consistent markup, styles, and behavior in prototype and production work.

## Frameworks


#### Plain HTML

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@amalitech-design-system/core@<VERSION NUMBER>/dist/amalitech-design-system/amalitech-design-system.css" />
<script type="module" src="https://cdn.jsdelivr.net/npm/@amalitech-design-system/core@<VERSION NUMBER>/dist/amalitech-design-system/amalitech-design-system.esm.js"></script>
<script nomodule src="https://cdn.jsdelivr.net/npm/@amalitech-design-system/core@<VERSION NUMBER>/dist/amalitech-design-system/amalitech-design-system.js"></script>
```

> Replace `@<VERSION NUMBER>` with your desired version

> Omit the `@<VERSION NUMBER>` to get the most recent version


#### With a bundler or ES modules

```js
import '@amalitech-design-system/core/dist/amalitech-design-system/amalitech-design-system.css';
import { defineCustomElements, applyPolyfills } from '@amalitech-design-system/core/loader';

applyPolyfills().then(() => {
  defineCustomElements(window);
});
```

## Utilities

#### Bundler

Import the output css in the root of your stylesheet after installation to get started with using our utilities.

```css
/* From main.css */
@import "/node_modules/@amalitech-design-system/utilities/dist/main.css";
```

#### CDN

```html
<link
    rel="stylesheet"
    href="https://cdn.jsdelivr.net/npm/@amalitech-design-system/utilities@<VERSION NUMBER>/dist/main.css"
/>
```

> Replace `@<VERSION NUMBER>` with your desired version

> Omit the `@<VERSION NUMBER>` to get the most recent version

## Design Tokens

#### Bundler

Import the output css in the root of your stylesheet or import the javascript/typescript variables in your file to get started with using our design tokens

```css
/* From main.css */
@import "/node_modules/@amalitech-design-system/design-tokens/dist/design-tokens-amalitech.css";

body {
  /* ----------  Main Colors -------------- */
  --primary: var(--amalitech-color-primary);
  --secondary: var(--amalitech-color-secondary);
  --dark: var(--amalitech-color-dark);
  --light: var(--amalitech-color-light);
}
```

```ts
import { color } from '@amalitech-design-system/design-tokens/dist/design-tokens-amalitech';

const primaryColor = color.primary;
const secondaryColor = color.secondary;
const darkColor = color.dark;
const lightColor = color.light;

```

#### CDN

```html
<link
    rel="stylesheet"
    href="https://cdn.jsdelivr.net/npm/@amalitech-design-system/design-tokens@<VERSION NUMBER>/dist/design-tokens-amalitech.css"
/>
```

> Replace `@<VERSION NUMBER>` with your desired version

> Omit the `@<VERSION NUMBER>` to get the most recent version
