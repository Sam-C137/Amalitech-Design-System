# Getting started for designers

Amalitech Design System provides a comprehensive component library that enables you to easily create brand-compliant, intuitive, and accessible products and services.

## 1. Install `Work Sans`

The following font is from Google. It is open source and may be used freely, created by [Wei huang](https://fonts.google.com/?query=Wei%20Huang)

[Download `Work Sans`](https://fonts.google.com/specimen/Work+Sans/)
