# How to use `amalitech-breadcrumb`

The breadcrumb component provides a visual representation of the user's path through the application, allowing them to easily navigate back to previous pages or sections.

You could use it like this:

## Default

<Canvas of={BreadcrumbStories.Arrow} />

```html
<amalitech-breadcrumb separator="arrow">
  <amalitech-breadcrumb-item href="https://google.com/" target="_blank">Home</amalitech-breadcrumb-item>
  <amalitech-breadcrumb-item active href="/">Users</amalitech-breadcrumb-item>
</amalitech-breadcrumb>
```

## Separators

The breadcrumb component comes with three main types of separators: arrow, dot and slash.
You can specify this by setting the `separator` attribute

### Arrow Separator

## Styling with CSS

Custom styling can be applied with exposed CSS variables

```html
<head>
  amalitech-breadcrumb { --breadcrumbSize: 2rem; --breadcrumbFont: "Work Sans", sans serif; }
</head>
<body>
  <amalitech-breadcrumb separator="slash" size="small">
    <amalitech-breadcrumb-item href="https://google.com/" target="_blank">Home</amalitech-breadcrumb-item>
    <amalitech-breadcrumb-item active href="/">Users</amalitech-breadcrumb-item>
  </amalitech-breadcrumb>
</body>
```

## All CSS Variables

```css
amalitech-breadcrumb {
  --breadcrumbSize: 2rem;
  --breadcrumbFont: 'Work Sans', sans serif;
}
```
