# How to use `amalitech-button`

The button component comes pre-styled with options for custom styling with props or css variables

You could use it like this:

## Attributes

### `type`

Attaching a type attribute to the `amalitech-button` element is similar to the native html button `type` attribute

```html
<form>
    <input type="text">
    <amalitech-button type="submit">Submit<amalitech-button>
</form>
```

### `disabled`

The `disabled` attribute is used to disable the button

```html
<amalitech-button disabled>Click Here</amalitech-button>
```

## Listening for events

You can listen for a click event on the `amalitech-button` element

```html
<body>
  <amalitech-button>Click Me</amalitech-button>
</body>
<script>
  const button = document.querySelector('amalitech-button');
  button.addEventListener('click', e => {
    console.log(e, 'clicked');
  });
</script>
```

## Styling with CSS

Custom styling can be applied with exposed CSS variables

```html
<head>
    amalitech-button {
        --amalitech-button-color: violet;
        --amalitech-button-bg: #000;
        --amalitech-button-font-size: 2rem;
        --amalitech-button-radius: 8px;
        --amalitech-button-ptb: 10px;
        --amalitech-button-plr: 20px;
    }
</head>
<body>
    <amalitech-button>Click Me</amalitech-button>
</body>
```

## All CSS Variables

```css
amalitech-button {
    --amalitech-button-color: #fff;
    --amalitech-button-bg: #000;
    --amalitech-button-font-size: 1rem;
    --amalitech-button-radius: 4px;
    --amalitech-button-ptb: 8px;
    --amalitech-button-plr: 16px;
    --amalitech-button-focus-color: orange;
}
```
