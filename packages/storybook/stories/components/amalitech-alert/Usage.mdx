# How to use `amalitech-alert`

The alert component is allows you to show alert messages to users

You could use it like this:

## Default

<Canvas of={AlertStories.Normal} />

```html
<amalitech-alert>
    <amalitech-alert-content>
        <div slot="title">...</div>
        <div slot="subtitle">...</div>
    </amalitech-alert-content>
</amalitech-alert>
```

## Showing the alert component

To trigger the alert component, you need to trigger a custom event on the window with 
with a detail which contians the title, subtitle and state of the alert, and a `target` 
which matches the `name` attribute value, set to the alert component.

```html
<body>
    <amalitech-alert name="demo-notification-manager">
        <amalitech-alert-content>
            <div slot="title">...</div>
            <div slot="subtitle">...</div>
        </amalitech-alert-content>
    </amalitech-alert>

    <amalitech-button>Alert Me</amalitech-button>

    <script>
        const alert = document.querySelector('amalitech-alert');
        const button = document.querySelector('amalitech-button');

        button.addEventListener('click', () => {
            window.dispatchEvent(
                new CustomEvent('alert:alert-notification', {
                detail: {
                    target: 'demo-notification-manager',
                    title: 'Welcome',
                    state: 'info',
                    subtitle: 'This is Amalitech Design System',
                },
                }),
            );
        });
    </script>
</body>
```


## States

This component has four main state: success, info, warning and error. Specify the state
by setting it to the `state` attribute.

### Success state

```html
<amalitech-alert>
    <amalitech-alert-content state="success">
        <div slot="title">...</div>
        <div slot="subtitle">...</div>
    </amalitech-alert-content>
</amalitech-alert>
```

### Info state

```html
<amalitech-alert>
    <amalitech-alert-content state="info">
        <div slot="title">...</div>
        <div slot="subtitle">...</div>
    </amalitech-alert-content>
</amalitech-alert>
```

### Warning state

```html
<amalitech-alert>
    <amalitech-alert-content state="warning">
        <div slot="title">...</div>
        <div slot="subtitle">...</div>
    </amalitech-alert-content>
</amalitech-alert>
```

### Error state

```html
<amalitech-alert>
    <amalitech-alert-content state="error">
        <div slot="title">...</div>
        <div slot="subtitle">...</div>
    </amalitech-alert-content>
</amalitech-alert>
```

## Position

Changes the `position` of the alert component

```html
<amalitech-alert position="top-right">
    <amalitech-alert-content>
        <div slot="title">...</div>
        <div slot="subtitle">...</div>
    </amalitech-alert-content>
</amalitech-alert>
```
