# How to use `amalitech-map`

The map component can be used to show any geographical location on a map. 
It can be used to show a single location or multiple locations. 
It can also be used to locate the current user with the help of a dedicated button.

You could use it like this:

## Default

<Canvas of={MapStories.Default} />

```html
<amalitech-map></amalitech-map>
```

## With Custom Dimensions

You can resize the map component by adding width and height attributes.

<Canvas of={MapStories.WithCustomHeightAndWidth} />

```html
<amalitech-map width="50%" height="400px"></amalitech-map>
```

## With Custom Zoom Level

You can set the zoom level of the map component by adding a zoom attribute.

<Canvas of={MapStories.WithCustomZoomLevel} />

```html
<amalitech-map zoom="15"></amalitech-map>
```

## With Custom Latitude and Longitude

You can set the latitude and longitude of the map component by adding `latitude` and `longitude` attributes.

<Canvas of={MapStories.WithCustomCoordinates} />

```html
<amalitech-map latitude="4.894491592221901" longitude="-1.756363203403844"></amalitech-map>
```

## With Custom Popup Name

You can customize the name of popups on the map component by adding a `name` attribute.

<Canvas of={MapStories.WithCustomPopupName} />

```html
<amalitech-map latitude="5.6339642" longitude="-0.1514209" name="Amalitech Head Office Accra"></amalitech-map>
```

## Without Nearby Locations

You can disable the nearby locations list feature by adding a `hide-near-by-list` attribute.

<Canvas of={MapStories.WithoutNearByList} />

```html
<amalitech-map hide-near-by-list></amalitech-map>
```

## Markers

You can add markers to the map component by calling the `addMarker` method.

```javascript
  document.addEventListener('DOMContentLoaded', () => {
    const map = document.querySelector('amalitech-map') as any;
    map.addMarker(4.894491592221901, -1.756363203403844, 'Amalitech Head Office Takoradi');
  });
```

## Styling with CSS

Custom styling can be applied with exposed CSS variables

```html
<head>
    amalitech-map {
        --map-border-radius: 0.5rem;
        --map-coordinates-icon-color: #000;
        --map-coordinates-list-text-primary-color: #eee;
    }
</head>
<body>
    <amalitech-map></amalitech-map>
</body>
```
